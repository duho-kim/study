* 이진 탐색(Binary Search)

- 데이터가 정렬돼 있는 상태에서 원하는 값을 찾아내는 알고리즘
- 대상데이터의 중앙값과 목표값을 비교해 데이터 크기를 절반씩 줄이면서 탐색
- 원하는 데이터를 탐색하는 가장 일반적인 알고리즘

- 타깃 데이터 탐색
- 중앙값 비교를 통한 대상 축소 방식
- 시간 복잡도 : O(logN)

* 핵심
오름차순 기준

1. 현재 데이터셋의 중앙값을 선택(ex 정렬된 11개의 수 중 6번째) 
2. 중앙값 > 목표값일 때 중앙값 기준으로 왼쪽 데이터셋 선택
3. 중앙값 < 목표값일 때 중앙값 기준으로 오른쪽 데이터셋 선택
4. 1~3 반복하다가 중앙값 == 목표값일 때 탐색 종료
