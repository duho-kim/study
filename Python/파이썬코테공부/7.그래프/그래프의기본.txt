* 그래프(Graph)
- 노드와 에지로 구성된 집합
- 노드 : 데이터를 표현하는 단위, 에지 : 노드를 연결하는 선
- 트리도 그래프의 일종

- 유니온 파인드 : 그래프의 사이클이 생성되는지 판별하는 알고리즘
- 위상 정렬 : 노드들을 선형으로 정렬, 값이 유일하지 않다
     조건) 사이클이 없는, 방향(전후관계)이 있는 그래프
     ex) 수강신청 (수1 -> 수2)
  
[최단거리 알고리즘]
- 다익스트라 : S라는 시작점이 있고 다른 모든 노드로 가는 최단거리를 구하는 알고리즘
              단, 음수 간선은 있으면 안된다
              시작점 O, 음수간선 X
- 밸만-포드 : 시작점 O, 음수간선 O
- 플로이드-워셜 : 시작점 X 임의의 모든 노드쌍의 최단거리 구하기

- 최소 신장 트리 : 그래프에서 최소의 가중치 합으로 모든 노드를 연결
                  사이클 있으면 안되기 때문에 유니온파인드도 같이 구현
