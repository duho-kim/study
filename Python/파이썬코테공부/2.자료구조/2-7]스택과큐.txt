스택 : LIFO(Last In First Out)

원래 자료구조 push, pop
파이썬 append, pop

s[-1] : 마지막 데이터 단순 확인
pop   : 마지막 데이터를 빼서 확인

- 파이썬에서는 list를 이용하여 스택을 구현
- 깊이 우선 탐색 DFS에서 자주 사용
- 백트래킹 종류
- 재귀함수 알고리즘 원리와 일맥상통
--------------------------------------------

큐 : FIFO(First In First Out)
s.append(data)  ->  rear(새값) 값 값 값 값 값 front(지울값)  ->  s.popleft()

s[0] : 단순 확인

- 파이썬에서는 일반적으로 deque를 이용하여 큐를 구현
- 리스트로도 구현할 수 있지만 시간복잡도가 덱이 더 좋다 
- 너비 우선 탐색 BFS에서 자주 사용

우선순위 큐
- 값이 들어간 순서와 상관 없이 우선순위가 높은 데이터가 먼저 나오는 구조
- front에 항상 최댓값 또는 최솟값이 위치
- 뒤에 자세히 공부
