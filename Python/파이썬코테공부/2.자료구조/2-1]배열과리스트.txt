파이썬 list와 자료구조 list 약간 다르다

파이썬에서 list는 배열의 특성도 내포하고 있다

무엇이 다르고 어떻게 다른가?

자료구조의 배열
- 메모리의 연속공간에 값이 채워져 있는 형태

자료구조의 배열의 특징
1. 인덱스를 사용해서 값에 바로 접근 가능
2. 새로운 값을 삽입하거나 인덱스에 있는 값을 삭제하기 어렵다
   삽입삭제하려면 해당 인덱스 주변값을 이동시켜야함
3. 배열의 크기는 선언할 때 지정할 수 있고 한번 선언하면 크기변경 불가

자료구조의 리스트
- 값과 포인터를 묶은 노드라는 것을 포인터로 연결한 자료구조

자료구조의 리스트 특징
1. 인덱스가 없으므로 값에 접근하려면 Head 포인터부터 순서대로 접근
   값에 접근하는 속도가 느리다
2. 데이터 삽입삭제 연산속도가 빠르다
3. 선언할때 크기 지정하지 않아도 됨

파이썬 리스트
index 바로접근이 가능하고 가변적이다 == 너무 좋다
